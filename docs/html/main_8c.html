DESCRIPTION

This source file implements all the logic to prvide the dunctionality outline in the header file. This file is for the STM32L100 family. Port A1 is used as the input for the keypad, Port B is used to set-up the active-high state needed for the keypad, and Port C as the output for the LEDs. The LEDs display the last pressed key. Finally, timer TM10 is used to provide the PWM signal needed to control the DC-motor.
